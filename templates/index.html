Project

About This Project

End-to-end ML pipeline from raw data to deployed web application

ML Pipeline
1
Data Ingestion & EDA

Loaded 1000-row UCI dataset; explored score distributions, correlations, and feature impact.

2
Preprocessing & Feature Engineering

OneHotEncoding for categoricals, StandardScaler for numerics, avg_rw_score feature added.

3
Model Training — 9 Algorithms

Linear, Ridge, Lasso, KNN, Decision Tree, Random Forest, Gradient Boosting, XGBoost, CatBoost.

4
Hyperparameter Tuning

RandomizedSearchCV (30 iterations, 5-fold CV) on Random Forest — best model selected.

5
Flask Deployment

REST API with /predict and /metrics endpoints, deployed to Render.com via GitHub CI.

Tech Stack
Python 3.11 scikit-learn XGBoost CatBoost Pandas NumPy Matplotlib Seaborn Flask Gunicorn Bootstrap 5 Chart.js GitHub Render.com
